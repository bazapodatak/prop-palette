CSS Architecture

Dynamic variables.
Zero javascript styling.

Themes controlled entirely by registered @property CSS custom properties. Switching context changes a single data attribute on the root element.

Inspect Variables

4 Themes

Light, Dark, Solarized, and High-Contrast. Handpicked palettes, not inverted defaults.

@property

Variables registered with syntax color. Allows the browser to interpolate values for smooth transitions on theme switch.

Persistence

Selection saved to localStorage. System preference checked on first visit using prefers-color-scheme.

Fallbacks

If @property is unsupported, themes still switch instantly. Only the transition animation degrades.
